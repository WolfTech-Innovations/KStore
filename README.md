# KStore by WolfTech Innovations
<img width="1366" height="736" alt="image" src="https://github.com/user-attachments/assets/9925c2d0-a87c-47e5-b810-e5ba56d63f76" />

KStore is a simple, TV-friendly app store for Plasma Bigscreen. It allows users to easily fetch and install filtered Android applications (specifically streaming and remote-compatible apps) through Waydroid.

## Features

- **Filtered App Selection**: Only shows apps that are remote-compatible or focused on streaming, ideal for TV use.
- **Easy Installation**: One-click installation via Waydroid.
- **User-Friendly UI**: Built with KDE Kirigami for a seamless Plasma Bigscreen experience.
- **WolfTech Innovations Branding**: Premium feel for non-Linux users.

## Requirements

- Plasma Bigscreen
- Waydroid
- Qt 5/6 and Kirigami
